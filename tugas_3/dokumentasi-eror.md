| File                         | Deskripsi Error                                                          | Penyebab                                                      | Perbaikan                                              |
| ---------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------- | ------------------------------------------------------ |
| **form-daftar.php**          | DOCTYPE salah (`<DOCTYPE >`)                                             | Salah penulisan standar HTML                                  | Ganti menjadi `<!DOCTYPE html>`                        |
| **proses-pendaftaran-2.php** | Variabel `$sekolah` ditulis `sekolah` tanpa `$`                          | Syntax error PHP                                              | Perbaiki menjadi `$sekolah = $_POST['sekolah_asal'];`  |
| **proses-pendaftaran-2.php** | Query SQL menggunakan **VALUE**                                          | Syntax SQL salah, harus **VALUES**                            | Ganti menjadi `VALUES (...)`                           |
| **proses-pendaftaran-2.php** | Redirect ke `index.php`, tetapi file tidak ada sehingga 404 Not Found    | File index.php belum dibuat                                   | Membuat `index.php` untuk menampilkan status           |

| **koneksi.php**              | “No connection could be made because target machine actively refused it” | MySQL belum aktif saat koneksi dipanggil                      | Menyalakan MySQL di XAMPP                              |
| **struktur folder proyek**   | File tidak ditemukan saat dibuka via browser                             | Folder tidak berada di `htdocs` atau nama folder tidak sesuai | Memastikan folder berada di `C:\xampp\htdocs\tugas_3\` |
